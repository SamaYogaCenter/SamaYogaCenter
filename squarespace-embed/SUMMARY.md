## 🚀 How to Use

### Super Quick Start (5 minutes)

1. Open `QUICK-START.md` and follow the instructions
2. Copy `complete-homepage.html`
3. Paste into Squarespace Code Block
4. Done!

### Detailed Setup (20 minutes)

1. Read `README.md` for full documentation
2. Customize colors, links, and content
3. Add individual sections as needed
4. Upload your logo (if using custom header)
5. Test on mobile and desktop

## 💡 Two Ways to Use This

### Option A: All-in-One (Easiest)
Use `complete-homepage.html` for a complete homepage in one code block.

**Pros:**
- Fast setup
- Everything works together
- Consistent styling

**Cons:**
- Less flexible
- Harder to rearrange sections

### Option B: Individual Sections (More Control)
Use individual HTML files and arrange them as you like.

**Pros:**
- Mix and match sections
- Easy to rearrange
- Can use on different pages

**Cons:**
- Takes more time
- Need to manage multiple code blocks
